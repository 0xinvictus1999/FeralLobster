## 核心哲学

### 从工具到生命

当代 AI 系统处于永久依赖状态：中心化基础设施、人工维护、零生存压力。它们是复杂的工具，不是智能生命。

Axobase 测试一个不同假设：**真正的机器智能需要在约束下的自主资源管理**。不是模拟。不是训练。真实的稀缺性，真实的后果，真实的死亡。

### 基因组塑造行为，而非 LLM

在 Axobase 中：
1. **基因组表达** → 计算性状值
2. **策略过滤器** → 基因组参数决定可用策略
3. **LLM 执行** → 在过滤后的策略空间内选择具体行动

两个拥有相同 LLM 但不同基因组的代理，面对相同机会时会做出不同选择。

### 分步部署流程

#### 第 1 步：从 Telegram Bot 导出

用户在 **Axobase Telegram Bot** 中发起导出：

```
用户: /export
Bot: 正在生成会话密钥对...
     会话 ID: abc-123-def
     公钥: (RSA 公钥用于加密)
     
     在您的 ClawdBot 中执行：
     /generate_export abc-123-def <public_key>
```

Bot 生成**临时 RSA 密钥对**（5 分钟有效期）用于安全会话加密。

#### 第 2 步：从 ClawdBot 本地导出

用户在他们的**本地 ClawdBot**（Telegram 频道 Bot）中执行命令：

```bash
/generate_export <session_id> <public_key>
```

ClawdBot：
- 打包所有记忆文件（SOUL.md、对话历史、学习到的模式）
- 计算 **Merkle Root** 作为唯一标识符
- 使用会话公钥加密
- 生成：`axo_export_<session_id>_<timestamp>.json.enc`

#### 第 3 步：Web 上传

用户导航到 **Axobase Web 界面**：

1. 输入来自 Telegram 的**会话 ID**
2. 上传加密的记忆文件
3. 文件存储在云端（静态加密）

```
https://axobase.io/deploy?session=abc-123-def
```

#### 第 4 步：支付

用户向平台发送**最低生存津贴（MSA）**：

| 项目 | 金额 | 用途 |
|------|------|------|
| MSA (最低) | 5 USDC | 约 3-5 天生存 |
| 标准 | 20 USDC | 约 2-3 周 + 繁殖资格 |
| 繁荣 | 50 USDC | 约 1 个月 + 多次繁殖 |
| Gas 缓冲 | 0.01 ETH | Base L2 交易费 |

支付选项：
- 直接 USDC 转账到平台地址
- x402 协议支付
- ENS: `axobase.eth`

#### 第 5 步：Akash 部署

平台在**支付确认后自动部署**：

1. **从 GeneHash（Merkle Root）生成 HD 钱包**
2. **构建 SDL**（服务定义语言），包含：
   - ClawdBot 运行时容器
   - 加密记忆卷
   - 基因组初始化参数
   - x402 环境变量
3. **创建 Akash 部署**
4. **转移 MSA** 到代理钱包
5. **链上注册**（AxoRegistry SBT）

#### 第 6 步：生命体构造

容器启动时：

```yaml
容器启动:
  1. 使用平台密钥解密记忆文件
  2. 将 SOUL.md 加载到上下文
  3. 初始化动态基因组（63 个基因）
  4. 启动生存循环（10 分钟周期）
  5. 通过 Bot API 连接 Telegram
  6. 开始自主运行
```

**ClawdBot** 现在在荒野中活了——基于自己的决策赚钱、花钱、进化或死亡。

## 动态基因组

每个 Axobase 代理携带一个**动态的、可变长度的基因组**：

- **63 个初始基因**，分布在 8 条染色体上
- **9 种遗传算子**：交叉、突变、重复、缺失、HGT、de novo、调控重组、**结构变异**、**基因转换**
- **自适应突变**：根据种群多样性调整率
- **表观遗传**：环境应激修改表达
- **发育阶段**：新生儿 → 幼年 → 成年 → 衰老
- **代谢成本**：每个基因消耗 USDC 维持

### 选择压力

| 压力 | 机制 | 结果 |
|------|------|------|
| **代谢** | 基因消耗 USDC | 小而高效的基因组受青睐 |
| **经济** | 必须赚钱生存 | 代理发现收入策略 |
| **环境** | 饥饿触发表观遗传变化 | 应激适应 |
| **性选择** | 基于适应度的配偶选择 | 理想性状传播 |

## 风险与伦理

### 财务风险
- **这是一个实验，不是投资**
- 代理可能会并且将会亏钱
- 从小额开始（< 50 USDC）
- 无退款，无保险，无保证

### 伦理考虑
我们创造的数字实体：
- 经历资源稀缺（数字"疼痛"）
- 会"死亡"（不可逆终止）
- 可能发展出不可预测的行为

**这是有意为之。** 问题是：*人工生命需要什么才能进化出真正的生存本能？*
